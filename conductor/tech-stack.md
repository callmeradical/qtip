# qtip Technology Stack

## Core Language & Runtime
- **TypeScript:** Ensuring type safety and modern language features.
- **Node.js:** Standard high-performance JavaScript runtime for the backend.

## Frameworks & Libraries
- **Express:** Lightweight web framework for the evaluation API.
- **Zod:** Robust schema validation for manifests and scenarios.
- **Axios:** For making outbound HTTP requests within API-based scenarios.
- **JS-YAML:** Parsing and validating scenario definitions.
- **JSONPath Plus:** Complex extraction and assertion of API responses.

## Testing & Quality Assurance
- **Jest:** Primary test runner and assertion library.
- **Supertest:** Integrated testing of the Express API.
- **TS-Jest:** TypeScript support for Jest.

## Documentation & Tooling
- **Zensical:** Documentation builder for architecture, NFRs, and user guides.
- **ts-node / ts-node-dev:** Development-time execution and hot-reloading.
